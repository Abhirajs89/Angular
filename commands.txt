Project Setup
----------------------------------------------------------
1) Install Nodejs

2) Go to command prompt and run below command to download and install angular cli globally

npm install -g @angular/cli@latest

3) Go to directory where you want to create the angular project and run command to create the angular app

ng new angular-app

4) Go to newly created folder and run command to run the app at localhost:4200

ng serve

5) Install bootstrap 3 for css styling into the project locally 

npm install --save bootstrap@3

6) Create Components with CLI using below command

ng g c servers
